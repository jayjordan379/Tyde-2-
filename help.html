1. How to use the search engine

    2. The browser can be used in two ways;

    • Website browsing:

    This browser is capable of enabling the user to navigate through multiple website on the internet.

    All that is needed is to insert the URL address of where you want the browser to take you and the website will be opened.

    For example, "http://www.google.com". This URL address will tell the browser to take you to www.google.com which is the google website.

    The URL address should be written inside the provided input bar and then the search button is pressed to make the search.

    • Online searching:

    The browser also has an inbuild search engine that can be used for search purposes abit something like google.

    To access the search engine, simply just write the info in the input bar but ignoring the "http://" part. the browser will automatically know and send the information to the search engine selected for further review of results.

  2. Why are some websites not working in the browser?

    3. Websites such as google, ask.com and others are not able to work inside of another website. This is why the development of the browser used to open google and other websites full screen space though the user will no longer be able to use the browser services anymore till they return to the main browser.

    Even so, there are some websites and search engines that can really work inside of the browser such as Bing serch engine and yandex search engine, tubidy.com and others.

    Luckily in order to solve this, the development of the browser has a special feature in the MENU dropdown.

    All that has to be done is to click the menu icon at the top right corner of the browser and choose the Open in new tab option to open the site.

    However, this feature is only allowed in the home page.
